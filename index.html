Controls

How to play

  • Use your right-side hand in frame to aim the saber.
  • Point your index finger to steer the blade direction.
  • Use your left-side hand to thrust forward for a Force Push.
  • Stand a little back from the camera so both hands stay visible.

Heuristics

Zero-data rules

Blade anchor Wrist + index base blend
Blade direction Anchor to index tip vector
Push trigger Area delta + palm z velocity
Collision math Line-to-sphere distance test
Hand role memory Position + handedness + temporal stickiness

Build Notes

Why this stack

This prototype keeps everything in the browser so we can ship the complete interaction loop now. Later, the same heuristics can move into Rust or a game engine without changing the gameplay model.

System Blueprint

Capture, interpret, react, render

The prototype now runs the whole loop in one place: webcam feed, MediaPipe landmarks, heuristic gesture logic, collision tests, and game feedback in a single live browser canvas.

01

Camera Input

Requests webcam access and mirrors the feed into the arena so the interaction feels natural.

02

MediaPipe Hands

Tracks up to two hands and returns the landmark coordinates that drive the whole simulation.

03

Heuristic Engine

Calculates saber vectors, push deltas, cooldowns, and state labels with no training dataset.

04

Gameplay Layer

Spawns drones, tests collisions, applies impulses, and paints particles and bloom-style glow.
